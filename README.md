
# Apartments Project

## Description
Web application for viewing and managing apartments and employees. Apartment and employee data are loaded dynamically from a PostgreSQL database.

## Getting Started
1. Install dependencies:
   ```
   npm install
   ```
2. Start the server:
   ```
   npm start
   ```
3. Open the site in your browser:
   ```
   http://localhost:3000
   ```

## Features
- View list of apartments and employees
- Add apartment (with photo)
- Add employee (with photo)
- Dynamic data display

## Stack
- Node.js (Express)
- PostgreSQL
- Bootstrap 5

## Structure
- `index.html` — main page
- `app.js` — client logic
- `server.js` — server logic
- `app.css` — styles
- `uploads/` — uploaded images
